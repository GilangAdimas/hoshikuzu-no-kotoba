感情（かんじょう）- Perasaan & Emosi

1. Perasaan Dasar

嬉しいうれしいSenang / Bahagia
悲しいかなしいSedih
怒るおこるMarah
楽しいたのしいMenyenangkan
怖いこわいTakut
寂しいさびしいKesepian
疲れるつかれるLelah
とても嬉しいです。
Saya sangat senang.
今日は疲れました。
Hari ini saya capek.

2. Pola ～です vs ～と思います

Untuk menyatakan perasaan sendiri → langsung pakai bentuk biasa + です.

悲しいです。
Saya sedih.

Untuk menyatakan dugaan tentang orang lain → gunakan ～と思います.

彼は悲しいと思います。
Saya pikir dia sedih.

⚠ Tidak bisa langsung mengatakan perasaan orang lain tanpa penanda.


3. ～そうです (Kelihatannya)

彼は怒っていそうです。
Kelihatannya dia marah.
彼女は嬉しそうです。
Kelihatannya dia senang.

Digunakan berdasarkan apa yang terlihat.


4. Ingin / Harapan (～たい / ～ほしい)

～たいIngin melakukan
～ほしいIngin sesuatu
休みたいです。
Saya ingin istirahat.
新しいスマホがほしいです。
Saya ingin HP baru.

5. Emosi + ～てしまいました (Penyesalan)

約束を忘れてしまいました。
Saya tidak sengaja lupa janji.
怒ってしまいました。
Saya terlanjur marah.

Digunakan untuk menunjukkan penyesalan atau kejadian yang tidak diinginkan.


6. Dialog Singkat

A: どうしましたか。
B: ちょっと疲れています。
A: 大丈夫ですか。
B: はい、でも少し悲しいです。

7. ～すぎる (Terlalu...)

Digunakan untuk menyatakan sesuatu yang berlebihan.

Pola:

  • い-adjective → buang い + すぎる
  • な-adjective → + すぎる
  • Verb stem + すぎる
暑すぎます。
Terlalu panas.
食べすぎました。
Makan terlalu banyak.
緊張しすぎています。
Terlalu tegang.

Biasanya mengandung nuansa negatif.


8. ～ばよかった (Seandainya...)

Digunakan untuk menyatakan penyesalan tentang masa lalu.

Pola: Bentuk ば + よかった

もっと勉強すればよかった。
Seandainya saya belajar lebih banyak.
行かなければよかった。
Seandainya saya tidak pergi.

Nuansa kuat penyesalan.


9. ～てよかった (Syukurlah...)

Digunakan untuk menyatakan rasa lega atau hasil yang baik.

合格してよかった。
Syukurlah lulus.
話してよかったです。
Syukurlah saya berbicara.

Perbedaan:

  • ～ばよかった → menyesal
  • ～てよかった → lega / hasil baik

10. Perbedaan 楽しい vs 嬉しい

Kata Makna Situasi
楽しい（たのしい） Menyenangkan Situasi / aktivitas seru
嬉しい（うれしい） Bahagia / Senang Perasaan pribadi karena suatu hasil
パーティーは楽しかったです。
Pestanya menyenangkan.
プレゼントをもらって嬉しかったです。
Saya senang karena mendapat hadiah.

楽しい → pengalaman menyenangkan
嬉しい → emosi bahagia karena sesuatu terjadi
