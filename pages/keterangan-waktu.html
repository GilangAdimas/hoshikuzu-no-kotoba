Keterangan Waktu dalam Bahasa Jepang

1. Apa itu Keterangan Waktu?

Dalam bahasa Jepang, keterangan waktu digunakan untuk menjelaskan kapan suatu kegiatan terjadi. Keterangan waktu disebut juga 時間の表現（じかん の ひょうげん）.

Keterangan waktu menjawab pertanyaan seperti:

いつ？ → Kapan?
何時？ → Jam berapa?
何曜日？ → Hari apa?
何日？ → Tanggal berapa?

2. Contoh Keterangan Waktu

Contoh jam:

七時（しちじ） → Jam 7
三時半（さんじはん） → Jam 3:30

Contoh hari:

今日（きょう） → Hari ini
明日（あした） → Besok
昨日（きのう） → Kemarin

Contoh tanggal & bulan:

五日（いつか） → Tanggal 5
八月（はちがつ） → Bulan Agustus
2024年（にせん にじゅうよねん） → Tahun 2024

3. Penggunaan Partikel に

Sebagian besar keterangan waktu menggunakan partikel に.

七時に起きます。
Saya bangun jam 7.
日曜日に勉強します。
Saya belajar hari Minggu.
八月に日本へ行きます。
Saya pergi ke Jepang bulan Agustus.

4. Keterangan Waktu TANPA に

Beberapa kata waktu tidak memakai に:

Kanji Hiragana Arti
今日 きょう Hari ini
明日 あした Besok
昨日 きのう Kemarin
毎日 まいにち Setiap hari
今週 こんしゅう Minggu ini
今日行きます。
(Bukan 今日に)

5. Struktur Dasar Kalimat

Pola umum kalimat dengan keterangan waktu:

[Waktu] + に + [Tempat] + で + [Kata Kerja]
明日学校で勉強します。
Besok belajar di sekolah.

Memahami keterangan waktu sangat penting dalam percakapan sehari-hari dan termasuk materi dasar level N5.
